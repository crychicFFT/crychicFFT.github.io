接口与协议选型表

这张表用于快速判断某个场景该用哪类通信方式。后续会继续扩充细节。

接口 / 协议 典型信号 / 载体 距离 速度 是否多节点 典型应用
UARTTX / RX / GND短中低速通常点对点调试串口、模块通信、单片机通信
RS232DB9 / 单端信号较短中低速点对点传统工控设备、旧串口设备
RS485A / B / GND较远中速支持工控、BMS、仪表、Modbus RTU
CANCAN_H / CAN_L较远中高速支持汽车、设备控制、电机、传感器总线
I2CSCL / SDA短中低速支持板级传感器、IMU、OLED、EEPROM
SPISCLK / MOSI / MISO / CS短高速通过片选支持Flash、屏幕、ADC、无线模块
USBD+ / D- / VBUS / GND短到中中高速主从结构USB 转串口、相机、调试器
Ethernet网线 / RJ45较远高速网络化支持上位机、设备通信、局域网设备
TCP基于 IP 网络取决于网络中高速支持可靠命令、文件传输、稳定数据通道
UDP基于 IP 网络取决于网络高速低延迟支持状态广播、实时数据、允许丢包场景
Modbus RTU常基于 RS485较远中速主从仪表、BMS、传感器、工控设备
CANopen基于 CAN较远中高速支持驱动器、传感器、工业模块
MQTT基于 TCP/IP取决于网络中速发布订阅物联网、状态上报、远程监控
OPC UA基于网络取决于网络中速支持工业数据集成、监控系统、设备数据建模
