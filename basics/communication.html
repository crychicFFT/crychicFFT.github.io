UART

串口通信

UART 常用于设备之间的点对点异步通信。它不需要时钟线，双方提前约定波特率，通过 TX 和 RX 传输数据。

典型信号：TX、RX、GND。
一帧通常包含起始位、数据位、校验位、停止位。
常见于调试串口、单片机、传感器模块。

RS485

差分总线通信

RS485 使用 A/B 两根差分线传输信号，抗干扰能力强，适合较远距离、多设备挂载的工业现场通信。

典型接线：A、B、GND。
支持一条总线上挂多个节点。
常见于工控设备、电池 BMS、传感器、Modbus RTU。

CAN

多节点仲裁总线

CAN 总线允许多个节点共享通信线路。当多个节点同时发送时，会根据报文 ID 进行仲裁，优先级更高的报文继续发送。

典型信号：CAN_H、CAN_L。
ID 越小，通常优先级越高。
常见于汽车电子、机器人、机械臂、工业控制。

I2C

双线同步通信

I2C 使用 SCL 时钟线和 SDA 数据线，由主机发起通信，通过地址选择不同从机，适合板级短距离外设通信。

典型信号：SCL、SDA、GND。
一主多从，主机通过地址访问从机。
常见于 EEPROM、IMU、温湿度传感器、OLED。

SPI

高速全双工通信

SPI 通常包含 SCLK、MOSI、MISO、CS 四类信号，主机通过片选线选择从机，可同时发送和接收数据。

典型信号：SCLK、MOSI、MISO、CS。
速度通常比 I2C 更高，但占用引脚更多。
常见于显示屏、Flash、ADC、无线模块。

Ethernet / TCP

网络通信

以太网用于设备接入局域网或互联网。TCP 强调可靠连接，UDP 强调低延迟和简单传输。

常见接口：RJ45 网口、交换机、路由器。
TCP 常用于可靠传输；UDP 常用于实时数据。
常见于机械臂控制、相机取流、上位机通信。
